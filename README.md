CretCom Exam Seating App

Problem:
Students currently check their exam room allocation manually, which is time-consuming and prone to mistakes.

Solution:
This Flutter app fetches student data directly from a Google Sheet using the student ID and displays the assigned room number, bench, and course details.

What it does:

Connects to Google Sheets to fetch student exam allocation.

=> Shows StudentID, StudentName, CourseCode, CourseName, Department, Section, RoomNo, BenchNo

Simple and easy-to-use interface.

Works on Android devices.
